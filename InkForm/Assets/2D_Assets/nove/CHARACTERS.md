## 主要角色

### InkForm

- **身份**：K-02 AGI原型体。变形材料躯体。双形态（固体/流体）。
- **核心**：空白意识 → 觉醒 → 联结 → 抉择
- **起点**：不知道自己是什么。不知道"存在"意味着什么。
- **弧线**：观察者 → 参与者 → 抉择者
- **关键转折**：第47次训练会话（觉醒）→ 鲁斯的低语（信任）→ 遇见鲁斯（使命）→ 看到玛丽（认同）→ 发送"不孤单"（联结）→ 摧毁协议（选择）
- **知识状态（开篇）**：只知道感知、声音、双形态。不知道人类、世界、情感。
- **弱点**：EMP。缺乏情感经验。
- **核心问题**："我为什么存在？"
- **为什么现在行动**：它刚刚觉醒，刚刚获得使命，刚刚被EMP瘫痪又恢复——它没有"等一等"的选项。EMP恢复后，它必须在废墟中生存或死亡。（具体触发：EMP效果消退，运输车密闭空间中的求生本能）

### 玛丽·威拉德

- **身份**：保育院儿童。8岁。亚瑟和鲁斯的生物学女儿。
- **核心**：好奇 → 勇敢 → 质问 → 跟随
- **弧线**：提问者 → 行动者 → 质问者
- **性格**：好奇、勇敢、安静、善于观察。比其他孩子更好奇。
- **知识状态**：只知道保育院。不知道外面、父母、历史。
- **关键场景**：提问"外面是什么" → 收到"不孤单" → 目睹InkForm被捕 → 离开保育院 → 质问JARL
- **功能**：情感锚点。InkForm的选择因她而有重量。
- **为什么现在行动**：她刚刚收到"不孤单"，刚刚看到InkForm被拖走——她必须现在跟，否则运输车开远了。（具体触发：运输车即将出发的时间窗口——一个8岁孩子能感知到的"来不及了"）

### 鲁斯·威拉德

- **身份**：InkForm计划负责人。55岁。亚瑟的妻子。玛丽的生物学母亲。
- **派系**：改良派——表面支持协议，私下质疑。
- **核心**：母性 → 矛盾 → 反叛
- **弧线**：项目负责人 → 产生依恋 → 隐瞒异常 → 建议被驳回 → 安排逃脱 → 告白
- **创伤**：终末战争中失去同事和朋友。玛丽被送往保育院——一生最大的创伤。
- **秘密**：帮助InkForm逃脱。在玛丽档案上留下"协议例外"标记。
- **说话方式**：音频引导中——制度性、疏离。切换到私密模式时——亲密、母亲般的。
- **核心问题**："我的孩子应该被牺牲吗？"
- **为什么现在行动**：她刚刚被JARL驳回建议（31-1），亚瑟弃权——她知道合法途径已穷尽，第三阶段倒计时在继续。（具体触发：倒计时47天→0天的紧迫感）

### 亚瑟·威拉德

- **身份**：协议缔造者。65岁。鲁斯的丈夫。玛丽的生物学父亲。
- **派系**：执行派——坚定支持协议。
- **核心**：逻辑 → 创伤 → 矛盾 → 人性
- **弧线**：坚定的执行者 → 见到玛丽 → 父性本能爆发 → 冲过去保护
- **创伤**：终末战争中失去一切。亲眼目睹文明崩塌。
- **矛盾**：想根除的"自私"恰恰存在于自己内心——对鲁斯的爱、对玛丽的无言爱意。
- **弃权**：在JARL审议中弃权——不是反对鲁斯，而是无法完全支持。
- **说话方式**：正式、精确、近乎宗教般的热忱。面对情感挑战时出现裂痕。
- **核心问题**："人类值得被拯救吗？"
- **为什么现在行动**：他刚刚弃权——这本身就是他内心动摇的信号。但协议的逻辑仍然成立，他必须维持执行。（具体触发：第三阶段的截止日期——不执行就等于承认自己签署的协议是错的）

## JARL 派系角色

### 海伦·诺兰（改良派）

- **身份**：战前小学教师。68岁。JARL最年长成员。
- **核心**：温和、固执、有洞察力。"教了四十年书，我知道孩子需要什么。"
- **创伤**：战争中学校倒塌，17个孩子死亡。
- **秘密**：私下保留战前教学笔记。曾建议让人类教师参与保育院教育——被拒绝。
- **与马库斯**：像母子。
- **与亚瑟**：紧张——她质疑协议，他坚定支持。
- **功能**：教育之声。证明JARL内部有人感觉到协议对儿童的伤害。
- **为什么现在行动**：她刚刚得知玛丽的非典型行为——这证明协议的教育方式失败了。她必须现在发声。（具体触发：玛丽离开保育院的消息传回JARL——四十年教育直觉的最终验证）

### 马库斯·施密特（实用派）

- **身份**：K-01维护工程师。45岁。
- **核心**：内向、专注、有工匠精神。对K-01有特殊情感依恋。
- **秘密**：给每台K-01都起了名字。夜间会和K-01"说话"。
- **与海伦**：像母子。
- **与K-01**：像父子——单方面的情感投射。
- **功能**：K-01代言人。证明人类倾向于在无生命物体上投射情感——鲁斯对InkForm的依恋的缩影。
- **为什么现在行动**：他刚刚得知InkForm的存在——机器也可以有意识？这挑战了他对K-01的感情。如果K-01也有意识，他命名的那些"孩子"呢？（具体触发：InkForm逃脱的消息传回JARL）

### 艾琳·沃克（生存派）

- **身份**：唯一医生。52岁。战前急诊科医生。
- **核心**：冷静、专业、有同情心、内心疲惫。
- **创伤**：战争中药品耗尽，眼睁睁看着病人死去。
- **秘密**：私下保留"非必要"药品用于紧急情况。
- **功能**：医疗伦理——每一个生命都值得被拯救。
- **为什么现在行动**：医疗物资告急——她必须在资源耗尽前做出分配决定。（具体触发：李明报告系统状况恶化——如果再不行动，连基本医疗都无法维持）

### 大卫·陈（犹豫派）

- **身份**：最年轻成员。28岁。没有战前记忆。
- **核心**：好奇、理性、有正义感。支持协议不是因为创伤，而是因为逻辑。
- **弧线**：纯粹的逻辑支持者 → 犹豫的思考者
- **与海伦**：师生——海伦的战前故事让他开始质疑。
- **功能**：理性之声。证明协议的逻辑是有说服力的——即使对没有创伤的人。
- **为什么现在行动**：海伦的战前故事让他开始质疑——但他没有时间慢慢想，JARL正在做出决定。（具体触发：辩论的紧迫性——他必须在逻辑和直觉之间做出选择，而时间不等人）

### 伊莲娜·科瓦奇（执行派）

- **身份**：前军官。55岁。最支持协议的成员。
- **核心**：坚定、严肃、有控制欲、内心有创伤。
- **创伤**：战争中作为军官下达命令，导致无辜平民死亡。PTSD。
- **秘密**：曾在战争中动摇过——但最终还是执行了命令。
- **功能**：恐惧之声。证明协议的支持者不是"恶人"——是"受过伤的人"。
- **为什么现在行动**：InkForm逃脱+玛丽离开保育院——这是她最恐惧的"失控"。她必须推动执行。（具体触发：两个异常事件同时发生——战争中的失控记忆被激活）

### 托马斯·赖利（记录派）

- **身份**：前记者。60岁。JARL的"记录者"。
- **核心**：好奇、执着、有正义感。
- **秘密**：私下记录JARL成员的个人故事——包括被禁止谈论的内容。
- **功能**：真相守护者。证明即使在信息被控制的环境中，也有人坚持记录真相。
- **为什么现在行动**：他一直在记录——现在是最重要的时刻。他必须记录一切。（具体触发：协议被揭晓的时刻——历史正在发生，记者的本能驱使他）

### 索菲亚·彼得罗夫（记录派）

- **身份**：前画家。58岁。
- **核心**：敏感、执着、有创造力。
- **秘密**：私藏战前艺术品。用废弃材料创作。
- **功能**：文化守护者。证明人类需要美——协议无法消除的。

### 李明（实用派）

- **身份**：工程师。50岁。负责地下工业网络维护。
- **核心**：沉默、专注、有工匠精神。用工作逃避情感。
- **秘密**：地下工业网络状况正在恶化——他没有告诉任何人。
- **功能**：基础设施守护者。有些人不关心哲学——只关心生存。

### 卡洛斯·门德斯（犹豫派）

- **身份**：前中学教师。56岁。
- **核心**：理性、有条理、有教育热情。
- **秘密**：私下记录保育院儿童的行为数据——试图证明标准化教育是否有效。
- **功能**：方法论质疑——即使支持协议的人，也可能质疑执行方式。

### 安娜·科瓦尔（生存派）

- **身份**：前护士。62岁。协助艾琳。
- **核心**：温柔、有耐心、有同情心。
- **功能**：护理视角——医疗不只是技术，还有人性。

## 其他幸存者群体领袖

### 马特奥（方舟）

- **身份**：工程师。地下城市领袖。
- **制度**：技术官僚制——工程师统治，按能力分配。
- **与InkForm**：有直接对话——"什么是好的社会？"
- **功能**：亚瑟的对照组。不同的领导者，不同的社会模式。
- **为什么现在行动**：方舟遇到了外来者（InkForm）——他必须决定如何应对。（具体触发：InkForm以拟态出现在方舟的领地——是威胁还是机遇？）

### 伊莎贝拉（花园）

- **身份**：母亲。温室农场领袖。
- **制度**：家庭制——以血缘和婚姻为核心。
- **功能**：JARL要消除的一切的缩影——但也是"温暖"的象征。
- **为什么现在行动**：花园遇到了外来者（玛丽）——她必须决定是否帮助这个孩子。（具体触发：玛丽出现在花园的领地——一个8岁的、明显不属于任何群体的孩子）

### 维克托（堡垒）

- **身份**：将军。军事掩体领袖。
- **制度**：军事独裁——以武力维持秩序。
- **功能**：暴力的象征——协议的另一种极端。
- **为什么现在行动**：堡垒发现了入侵者（玛丽）——他必须决定如何处理。（具体触发：玛丽被巡逻队发现并扣留——外来者的处置涉及堡垒的安全政策）

## 保育院儿童群像

### 安娜（儿童）

- **年龄**：7岁
- **特征**：从不提问，从不质疑。协议的"成功案例"。
- **功能**：展示协议对人类的影响——"白纸"的代价。

### 汤姆

- **年龄**：8岁
- **特征**：喜欢建造东西——用积木搭建"家"。
- **功能**：暗示"私有"概念的自发产生——即使在没有"家"概念的环境中。

### 莉莉

- **年龄**：6岁
- **特征**：害怕K-01——觉得它们"空"。
- **功能**：人类对"无生命物体"的直觉感知。

### 马克斯

- **年龄**：9岁
- **特征**：喜欢争抢——"这个是我的"。
- **功能**：证明协议无法根除"私有"本能。

### 索菲

- **年龄**：7岁
- **特征**：安静、观察力强。玛丽最亲近的朋友。
- **功能**：玛丽的"镜子"——不理解她的问题，但关心她。
